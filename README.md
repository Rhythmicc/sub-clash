<h1 style="text-align: center"> sub-clash </h1>

## Install

```shell
pip3 install sub-clash -U
```

## Usage

```shell
sub-clash --help
```

## Developer

If you need use global config, just edit `__config__.py`:
1. make `enable_config = True`.
2. edit `questions` list.
3. using `config` at `main.py`.
